简单的JAVA程序

2018.07.16 郭小睿同学  热度 ℃

开发一个简单的Java程序

Java程序的类型

  • Application和Applet程序
    • 结构和运行环境不同
    • 前者是独立的程序，需要调用虚拟机来运行
    • 后者是嵌在HTML网页中非独立的程序
      • 由专门的appletViewer来运行
      • 或者由web浏览器（调用java虚拟机）来运行

Application程序

  • HelloWorldApp.java
  • 要点
    • class是主体
    • public类名与文件同名
    • main()的写法是固定的
    • System.out.print 及println 及 printf

Applet程序

  • HelloWorldApplet.java
    • import表示导入
    • extends JApplet表示继承
      • Applet 或 JApplet 都可以
    • 有paint()方法，表示如何绘制
    • 没有main()方法
  • HelloWorldApplet.html

Java程序的基本构成

  • HelloWorld.java
    • package语句 (0或1句)
    • import语句 (0或多句)
      • 导入其他类的类库
    • 类定义———class (1个或多个)
      • 一个java文件中只能有一个public类（且与文件同名）
  • 类=类头+类体
  • 类成员有成员变量和成员方法。
  • 方法=方法头+方法体

Applet的编辑、编译与运行

  • Java Applet程序必须嵌入到HTML中，并由负责解释HTML文件的WWW浏览器充当解释器，解释并执行程序
  • Java Applet在WWW中引入了动态交互的内容
  • 使用applet标签
  • 使用appletViewer运行Applet
    • appletViewer HelloWorldApplet.html

程序的输入与输出

  • 应用程序（Java Application）的输入输出可以是文本界面，也可以是图形界面。
  • 小程序（Java Applet）只能是图形界面。
